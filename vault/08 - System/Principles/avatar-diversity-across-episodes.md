# avatar-diversity-across-episodes

**Rule.** When a series uses an imagined listener / viewer to anchor each episode, vary the avatar across episodes. Don't default every episode to the same gender, region, age, family structure, or cultural register.

**Why.** A fictional listener gives a script its emotional centre — but if every episode imagines the same demographic, the writer's blind spots become the show's blind spots. A real audience is mixed. If the avatar pool reflects only one slice, the audience that doesn't match feels written-around-not-for, episode after episode. The fix is mechanical: before writing a new avatar, look at the recent ones and pick something different.

**How to apply.**
- Before sketching a new heartfelt-angle avatar, scan the last 3 episode notes' Heartfelt Angle sections.
- Vary at least two of: gender, region, age band, family structure, name's cultural register.
- The audience is mixed; the avatar pool should reflect that.
- The avatar's specifics shouldn't matter to the episode's main argument — if changing the avatar changes the script meaningfully, the script is over-coupled to one demographic.
- Codify the variety check as a phase task in the script-writer directive's checklist so it actually happens, not just gets remembered.
